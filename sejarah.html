Sejarah Java

Java pertama kali dirilis oleh Sun Microsystems pada tahun 1995. Proyek Java dimulai oleh James Gosling, Patrick Naughton, dan Mike Sheridan yang awalnya dikenal sebagai proyek "Green" dan fokus pada pengembangan teknologi untuk peralatan konsumen digital. Java berkembang menjadi bahasa pemrograman yang populer dan luas digunakan dalam berbagai aplikasi dan platform.
