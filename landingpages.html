Pengertian Java

Java adalah bahasa pemrograman tingkat tinggi yang banyak digunakan untuk pengembangan aplikasi berbasis desktop, web, mobile, dan lainnya. Java dikembangkan oleh Sun Microsystems pada tahun 1995 dan dirancang agar dapat dijalankan pada berbagai platform dengan moto "write once, run anywhere" (WORA).
